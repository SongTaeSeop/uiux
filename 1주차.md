### 웹 호환성

여러 기기에서 다양한 버전의 웹 브라우저로 웹페이지에 접속하기 때문에 웹 표준을 준수하는 것이 좋다. 웹 표준을 준수하면 다양한 버전에서 똑같은 사용자 경험을 보장할 수 있다.

- 공공기관의 경우, 보안 목적으로 여러 기능을 제거한 웹 브라우저를 기관에서 배포함
- 보통은 5년을 생명 주기로 봄, 현재 신기술은 5년 뒤에야 범용적으로 사용할 수 있음
- [W3C](https://www.w3.org/): 웹 호환성 표준 개발 조직
- 현재는 Chrome, Firefox, Safari 3개의 호환성을 확인하면 됨

### 정리

1. 웹 표준: 웹 호환성을 보장하기 위해 정한 기술, 약속, 규범
2. 웹 호환성: 모든 브라우저에서 동일하게 구현되는 페이지를 만드는 것

*웹 표준을 준수하지 않아도 페이지는 작동하지만,* 접근성 관련 법령에 의해 웹 표준을 준수하는 것이 일반적이다.
